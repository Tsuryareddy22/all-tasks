TAGS

The keyword which is enclosed b/w < and > to display the data in broweser, every tag its own behaviour

Types of tags

1.paired tag or container tag
The paired tag which is having opening and closing tag is known as paired tag
2.unpaired tag or empty tag
The tag which closes itself is known self closing tag or unpaired tag

Types of Elements

1.Block Level Element.
The tag which occupies whole width and creates a new line
Ex: h1h6, p, div, semantic elements
2. Inline-block Element.
The Elements which occupies only required space and does not create a n line and we can apply width, height, margin and padding properties
Ex: img, iframe, input, button
3. inline element.
The Elements which occupies only required space and does not create a ne line, in css we cannot apply width, height, padding top and bottom and margin to and bottom.
ex: span formatting tags, b, strong,i
